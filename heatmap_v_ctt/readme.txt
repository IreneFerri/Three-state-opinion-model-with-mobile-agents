El format dels fitxers és el seguent:
res(velocitat mínima)_(velocitat maxima)_(alpha).csv

El conjunt d'1, -1 i 0 son les orientacions en que s'ha arribat al consens
per aquella velocitat, alpha.

Dins de cada fitxer, a cada fila hi ha les mateixes condicions inicials 
(representa una repetició) per diverses velocitats. Així, cada columna és un
valor de la velocitat. A l'hora de fer el heatmap, es compta el nombre de 0
(consensos neutres) d'una columna determinada.
